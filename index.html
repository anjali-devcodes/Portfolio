Featured Project

DietBite

Indian Nutrition Intelligence Platform — my flagship, full-stack project.

DietBite is a web application built to help dietitians, nutritionists, and health-conscious users create meal plans, calculate nutrients, and manage patient nutrition using Indian food composition data — an area most mainstream nutrition software overlooks in favor of Western food databases.

RoleOwner / Full-Stack Developer
StatusActively improving
UsersAdmin · Dietitian · User
Repoanjali-devcodes/DietBite-portfolio
Problem

Most nutrition software is built around Western food databases, so Indian foods are missing or poorly represented. On top of that, manual nutrient calculation by hand is slow and error-prone for dietitians managing multiple patients.

Solution

DietBite brings food search, a live nutrient calculator, patient management, and meal planning into one Indian-focused platform — backed by IFCT-based food composition data and role-based access so dietitians and their patients see only what's relevant to them.

Core features

🔐

Secure Authentication

JWT-based login with access & refresh tokens, verified end-to-end through Swagger UI.

🧑‍🤝‍🧑

Role-Based Access

Separate Admin, Dietitian, and User experiences with permission boundaries manually tested.

📋

Patient Management

Dietitians track patients with ownership isolation, BMI, and measurement history.

🔍

Indian Food Search

Filterable search across an IFCT-based database — currently 43 verified foods and growing.

🧮

Nutrient Calculator

Build a meal, and see combined macro & micronutrient totals with %RDA instantly.

🗓️

Meal Plan Builder

Multi-day meal planning with a searchable food picker and per-day nutrition averages.
